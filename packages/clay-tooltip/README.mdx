Simplest way of using Tooltip is by leveraging it's `show` prop and specifying `alignPosition` to determine it's position relative to the element it's tied to.

By using the [TooltipProvider](#tooltipprovider) wrapper you can bring the control over the tooltip to the component that is tied to it.

<Tooltip />

## TooltipProvider

By using the `<TooltipProvider>` it allows you to add the desired tooltip content on each element as a data attribute.

Here's a list of attributes the TooltipProvider can leverage:

-   `title` is for the tooltip content.
-   `data-tooltip-align` is for alignment direction.
-   `data-tooltip-delay` is for the delay(ms) before showing the tooltip.

<TooltipProvider />

### contentRenderer

`contentRenderer` prop gives you the ability to format the content passed to the Tooltip via the `title` attribute to meet your requirements.
